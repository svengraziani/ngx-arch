Process Pattern

The Process Pattern is a pattern that provides additional meta information about an observable stream.

It decorates the original data stream and wraps it into a simplistic pattern.

Example

In the context of an "isLoading$" stream (Observable[boolean]), the primitive boolean does not provide enough information to handle an exact UI representation.

An HTTP Call may not only be loading or not-loading. For example, a stream could have errored or never started.
